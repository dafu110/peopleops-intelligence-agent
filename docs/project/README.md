# Project Notes

This directory keeps local planning and delivery notes out of the repository root.

| File | Purpose |
| --- | --- |
| `task-plan.md` | Historical phase plan from the earlier UI optimization work. |
| `findings.md` | Research notes and design decisions from earlier discovery. |
| `progress.md` | Session progress log and validation notes. |

These files are not runtime inputs. They are retained for auditability and future handoff context.
